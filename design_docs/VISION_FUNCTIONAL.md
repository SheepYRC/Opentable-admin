# OpenTable: 愿景与功能规格

## 1. 产品愿景
"本地数据IDE" —— 打造一个既能让业务人员轻松上手，又能让技术人员深度定制的本地数据管理与分析平台。

- **核心名称**：OpenTable
- **设计哲学**：复杂选配化（按需展示复杂度）、本地优先、AI 原生。

## 2. 功能三级跳 (Product Phases)

### Phase 1: 极速查看器 (The Viewer)
- **秒开大文件**：支持 GB 级 CSV/Excel/Parquet 拖拽即看。
- **兼容 类 Excel 交互 和 SQL查询**：高性能虚拟滚动表格，支持基础过滤、统计，兼容 SQL 查询。
- **内存暂存 (Draft Mode)**：修改不直接入库，提供类似 Git 的“修改/保存”心智。

### Phase 2: 数据资产与自动化 (Data Assets & Automation)
- **多源挂载与同步**：实现文件夹监听、API 定时抓取，构建动态更新的本地数据集。
- **精细化导入策略**：深度引导式导入配置，支持自定义 JS 预处理算子与复杂类型纠错。
- **逻辑配方 (Recipes)**：将多步清洗与转换操作保存为可复用的“配方”，实现生产级的流水线处理。

### Phase 3: 智能增强与深度洞察 (Intelligent Insights)
- **AI 交互引擎 (NLP)**：通过 Text-to-SQL/Action 驱动复杂的清洗、合并及关联查询。
- **语义探索 (LanceDB)**：集成向量检索，支持基于业务语义的自然语言数据查找。
- **智能分析洞察**：AI 自动挖掘数据背后的规律，生成具有前瞻性的业务报表与交互式看板。
